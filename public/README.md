# Prairie Robotics Alliance — Public Documents

This directory contains documents made publicly available by Prairie Robotics Alliance, Inc.
Files placed here are served directly from the website at `/public/`.

## Directory Structure

```
public/
├── determination-letter.pdf       ← IRS 501(c)(3) determination letter (place here when received)
├── 990/
│   ├── 990-2025.pdf               ← Form 990 for fiscal year 2025 (place here when filed)
│   └── ...                        ← Future years
└── governance/
    ├── articles-of-incorporation.pdf   ← Filed articles of incorporation
    ├── bylaws.pdf                      ← Current bylaws
    └── ...                            ← Other governance documents
```

## Notes

- All documents in this directory are publicly accessible.
- Do NOT place any documents here that contain sensitive information (SSNs, passwords, financial account numbers, etc.).
- For the IRS determination letter: once received, rename the file to `determination-letter.pdf` and place it in this folder, then update the link in `index.html`.
- Form 990s are due 4.5 months after fiscal year end (or 7.5 months with extension). Post them here after filing.
- Written acknowledgment letters for donors ($250+) are NOT posted here — those are sent directly to donors.
